NextFlow — Node Contracts

Defines the inputs, outputs, and connection compatibility rules for every node in the canvas. An edge is only valid when the source output type matches the target handle's expected type.

Output Types

Text
A plain UTF-8 string
Image
A public URL pointing to an image
Video
A public URL pointing to a video

Compatibility Matrix

Source output → Text input Image input Images[] input Video input
Text ✓ ✕ ✕ ✕
Image ✕ ✓ ✓ ✕
Video ✕ ✕ ✕ ✓

Images[] is a multi-connection handle — multiple Image outputs can connect to it and are collected into an array at runtime.

Nodes

T

TextNode

Provides a static text value to downstream nodes

Inputs
text Text User — textarea optional
Output — handle: output
Text The exact string entered by the user
🖼

UploadImageNode

Uploads a local image and exposes its CDN URL

Inputs
file File User — file picker (PNG, JPG, WEBP) required
Output — handle: output
Image CDN URL of the uploaded image (Transloadit)
▶

UploadVideoNode

Uploads a local video and exposes its CDN URL

Inputs
file File User — file picker (MP4, MOV, WebM) required
Output — handle: output
Video CDN URL of the uploaded video
✦

RunLLMNode

Sends a prompt to a Gemini model, returns text

Inputs
model Enum User — dropdown only (not connectable) optional
system_prompt Text User textarea or connected Text node · handle: system_prompt optional
user_message Text User textarea or connected Text node · handle: user_message required
images Image[] Connected Image nodes (multiple) · handle: images optional
Output — handle: output
Text The LLM's text response
✂

CropImageNode

Crops an image to a percentage-based region

Inputs
image_url Image Connected Image node · handle: image required
x_percent Number User — number input (0–100, default 0) optional
y_percent Number User — number input (0–100, default 0) optional
width_percent Number User — number input (0–100, default 100) optional
height_percent Number User — number input (0–100, default 100) optional
Output — handle: output
Image URL of the cropped image
⏱

ExtractFrameNode

Extracts a video frame at a timestamp as an image

Inputs
video_url Video Connected Video node · handle: video required
timestamp Text User — text input (e.g. "5" or "00:01:30", default "0") optional
Output — handle: output
Image URL of the extracted frame
